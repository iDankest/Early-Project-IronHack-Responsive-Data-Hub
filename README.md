### 📋 Resumen del Proyecto
Este proyecto es la culminación de la primera fase del Bootcamp de **Ironhack**. El objetivo principal es construir una plataforma web totalmente **adaptable** que gestiona y visualiza datos de forma dinámica mediante una arquitectura de "Data Hub" simulada con una API interna (JSON).

---

### 🛠️ Características Principales
- **Inyección Dinámica de Datos:** Consumo de una API interna mediante `fetch` para renderizar tarjetas de contenido sin escribir HTML estático.
- **Vistas de Detalle:** Sistema de navegación que permite profundizar en la información específica de cada recurso del JSON.
- **Maquetación Adaptable (Responsive):** Diseño optimizado para dispositivos móviles, tablets y escritorio utilizando **CSS**.
- **Validación de Formularios:** Gestión de entrada de usuario con feedback inmediato y manejo de eventos en consola.

---

### 💻 Stack Tecnológico
* **Estructura:** HTML5 Semántico.
* **Estilos:** CSS (Maquetación ágil y moderna).
* **Lógica:** JavaScript ES6+ (Manipulación del DOM y Fetch API).
* **Diseño:** Prototipado en Figma.
